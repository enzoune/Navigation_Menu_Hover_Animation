# 🚀 Animated Navigation Menu (NavBar Animada)

> Componente de navegação moderno com foco em micro-interações e transições fluidas, desenvolvido puramente com HTML5 e CSS3.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir uma **NavBar Animada** que utilize as melhores práticas de semântica HTML para criar uma base robusta para animações de interface (UI).

O foco do desenvolvimento foi a experiência do usuário (UX), garantindo que a navegação principal (Início, Sobre, Serviços e Contato) seja apresentada de forma clara e visualmente engajadora.

## ✨ Destaques e Diferenciais Técnicos

* **Semântica HTML5 Pura:** Uso correto das tags `<nav>` e `<ul>` para garantir uma estrutura acessível e compreensível por motores de busca e leitores de tela.
* **Micro-interações de Interface:** Estrutura preparada para efeitos de *hover* e transições dinâmicas, demonstrando atenção aos detalhes estéticos e funcionais.
* **Arquitetura Modular:** Uso de um contêiner principal (`.container`) que encapsula a lógica de navegação, facilitando a portabilidade do componente para diferentes projetos.
* **Design Clean e Minimalista:** Foco na hierarquia visual, eliminando ruídos e priorizando o conteúdo de navegação essencial.
* **Organização Profissional:** Separação rigorosa de responsabilidades, mantendo a estrutura no `index.html` e a identidade visual em uma folha de estilos dedicada (`css/style.css`).

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Marcação estrutural e semântica de navegação.
* **CSS3:** Gerenciamento de layout, tipografia e animações de transição.
